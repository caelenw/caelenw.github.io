Assignments

Assignment 1 - Basic HTML

Learning the basic fundamentals of web design using basic HTML. Learning to create titles, paragraphs, headings, and subheadings.

Assignment 2 - Basic CSS

Create a main page with custom fonts, colors, and a styled layout. Made a new page with a title, navigation, content, images, and a table, all featuring consistent and appealing styling.

Assignment 3 - Page Layout

Using Flexbox for layout, including a nav bar and styled content sections. It ensures the design flows well to different screen sizes and looks visually appealing.

Assignment 4 - Recreate CSS Page

Recreation of CSS home page, using flex to properly display content on various size screens. In addition to learning to copy styles and layout.

Assignment 5 JavaScript Buttons, Functions, and More

Implementing Button's with functions to implement

Assignment 6 Conditionals

Learning Functions to switch page veiws displaying different content without leaving the page. Using if statements for color and picture size to correctly display

Assignment 7 Loop

Used JavaScript and loops to create a random star display. In addition to displaying stars at random there is the ability to click and veiw the stars index.

Assignment 8 Arrays

Used arrays within JavaScript to properly display titles that correspond to an array index. Additinoally added picture descriptions that appear below the title.

Assignment 9 Classes

Implementing classes for different types of birds. On click of any given bird section a model window will display with details as well as an image
